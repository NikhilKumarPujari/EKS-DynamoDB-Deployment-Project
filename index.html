⚙️ DevOps Architecture

This project demonstrates a Cloud Native DevOps pipeline.

🐳 Application is packaged as a Docker container.

📦 Image is pushed to AWS ECR.

☸️ Container is deployed to AWS EKS (Kubernetes).

🌐 Traffic is exposed using a Load Balancer.

🗄️ User data is stored in AWS DynamoDB.

🚀 This architecture ensures scalability, high availability and cloud native deployment.
